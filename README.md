# Xrdp Install Script
Auto xrdp installer
Created By Horapusa

## AVAILABLE ON :

* Termux
### TESTED ON :

## How To Install
* `sudo su`
* `git clone https://github.com/bhathiya2005/Xrdp`
* `cd Xrdp`
* `ls`
* `chmod +x *`
* `bash xrdp.sh`
