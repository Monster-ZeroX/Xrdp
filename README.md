# Xrdp Install Script
Auto xrdp installer
Created By Horapusa

## AVAILABLE ON :

* Ubuntu
### TESTED ON :
* Ubuntu 18.04

## How To Install
* `sudo su`
* `git clone https://github.com/bhathiya2005/Xrdp`
* `cd Xrdp`
* `ls`
* `chmod +x *`
* `bash xrdp.sh`
* Unlock Port 3389 In your Vps Firewall.

## How To Connet
#### Windows
Use Your Windows remote desktop software.
#### Android 
Download the RD client app from the play store.
